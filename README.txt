1) Install Visual Studio Code
2) Install Arduino 1.18.9 Version. Not 2.x
3) Install Arduino, C/C++, C/C++ Extension Pack extensions from vscode
4) Add this line to your settings.json file (dependent on where you installed Arduino) "arduino.path": "C:\\Program Files (x86)\\Arduino"
5) Click Select Board in bottom right of vscode
6) Select "Arduino Uno"(will need to confirm with team)
